# Monitoring Server

## Tentang Tugas
Tugas yang di buat pada final project kali ini adalah menginstall php monitoring server dengan php versi 7.4.

## OS Server
CentOS 7.9

## Service Yang Digunakan
1. SSH Server
2. DHCP Server
3. Apache2
4. MariaDB
5. PHP Versi 7.4
6. FTP Server menggunakan FileZilla
